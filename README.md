# Early or Late — Validation Prototype

A single-page concept validation experience for the Early or Late wearable-art brand.

## What this is

A static HTML survey that doubles as a brand experience. Visitors land, engage with 8 embedded survey questions, and either claim early access or skip out — all behaviour and answers captured for the validation scorecard.

## Live site

Deployed via Netlify. Pushes to `main` auto-redeploy.

## Data capture

- **Survey data:** Netlify Forms — captures the structured submission on sign-up. View in Netlify dashboard → Forms.
- **Behavioural data:** Microsoft Clarity — session replay, heatmaps, rage clicks. View at clarity.microsoft.com.

## Local development

It's a single HTML file. Open `index.html` in a browser. No build step.

## Stack

- HTML / CSS / vanilla JS (no framework)
- Inter (typography), Helvetica fallback for the wordmark SVGs
- Microsoft Clarity (behavioural analytics)
- Netlify Forms (structured data capture)
- Netlify (hosting)
